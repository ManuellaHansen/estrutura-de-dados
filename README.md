# Exercícios de Lógica de Programação: Estruturas de Vetores

Este repositório contém a resolução de quatro algoritmos desenvolvidos em **JavaScript**. 


## Exercícios realizados 

### 1️⃣ Soma de Números Pares
* **Descrição:** O programa lê 10 números inteiros informados pelo usuário, armazena-os em um vetor e, ao final, calcula e exibe a soma apenas dos números que são pares.
* **Conceitos Aplicados:** Laço `for`, Operador de resto da divisão (`%`) para descobrir números pares, Acumulador de soma.

### 2️ Vetor com 7 Números Aleatórios
* **Descrição:** Preenche automaticamente um vetor de 7 posições com números gerados aleatoriamente pelo computador e exibe os valores na tela de uma só vez.
* **Conceitos Aplicados:** Laço `for`, Geração de números com `Math.random()` e arredondamento matemático para números inteiros com `Math.floor()`.

### 3️ Contagem de Ocorrências
* **Descrição:** Solicita 10 números ao usuário para preencher um vetor. Em seguida, pede um número adicional para busca e conta exatamente quantas vezes esse número aparece dentro do vetor digitado.
* **Conceitos Aplicados:** Preenchimento de vetor, Entrada de dados sequencial, Estrutura de varredura (busca) com laço de repetição e contador.

### 4️ Remover Valores Duplicados
* **Descrição:** O usuário digita 10 números que são salvos em um vetor. O algoritmo analisa esses dados e cria um segundo vetor contendo apenas os números inéditos (removendo os duplicados). Exibe o vetor original e o vetor limpo na tela.
* **Conceitos Aplicados:** Laços de repetição aninhados (um `for` dentro de outro `for`), Controle de estado com variáveis booleanas (`true`/`false`) e manipulação manual de índices de vetores diferentes.
